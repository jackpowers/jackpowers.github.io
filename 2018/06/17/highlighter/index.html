A Selenium Web Element Highlighter in C#

A Selenium Highlighter C Sharp

Back in the day, Selenium had a feature that would allow for highlighting web elements. This feature was useful for debugging, and it also made for more interesting viewing of test runs in the browswer. Alas, no more. So in this post, I’m making a simple implementation of a Selenium highlighter.

I built this app using Visual Studio Community Edition. Download it if you haven’t already.

Follow the steps below to build your own Selenium Highlighter in C Sharp

1. Create a new MSTest Solution

Open Visual Studio and select File -> New Solution. In the pop-up window, select .Net Core -> Tests -> MSTest Project. Create a name for your project (I’m using TableTraverseHighlightCSharp, so use this name if you want to copy and paste code.), and select a directory on your computer to store it.

2. Add Packages

Select Project -> Add NuGet Packages. Type in “Selenium” in the search field. Check the checkboxes next to Selenium.Webdriver and Selenium.Support. Click the Add Packages button and then accept the licenses in the pop-up window.
