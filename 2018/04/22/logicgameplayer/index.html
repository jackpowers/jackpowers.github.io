A Selenium Game Player

Today’s post is about a pretty cool two-part project I’ve been working on. The first part of the project was creating an educational strategy game, The Logic Game, to help logic students learn the truth-table definitions of logical operators in propositional calculus. The second part was creating a Selenium bot to play the game and report who won.

Part 1: The Logic Game





A Selenium bot plays the Logic Game



Before I got into professional software development, I was a philosophy professor. I used to teach courses in propositional logic. One of the biggest challenges for students in the course was learning the truth table definitions of the logical operators. I read about a board game that some philosophers came up with to gamify learning these definitions. The game seemed super cool, so I thought I’d build a digital version.

It works like this: first, the computer randomly generates an initial set of truth values to start the game. Then, players take turns selecting logical operators to set subsequent truth values. At the cost of a turn, a player can choose to use a negation tilde change an established truth value.






One player plays for True, and one player plays for False. The goal of the game is to make the biconditional triple-bar at the bottom of the board come out True if you are playing for True, and False if you are playing for False; play ends and a winner is declared when either of these events happen.





For a complete account of the game and its rules, look here. You can play the game here.

Part 2: Selenium Player Bot

Long story short, I used Java and Selenium to create a bot that will…

1) Open up a Chrome browser
2) Navigate to the Logic Game web page
3) Click on the DOM elements necessary to play the game to completion, and
4) Report whether True or False won in the console.

You can check out the code and clone the repo for this one here.

File Structure and Main

Nothing too complicated here. I have a main class and a game player class in two Java files. The most complicated bit is all the dependencies. You can check out the file structure and dependencies in more detail here.





The LogicGamePlayer class and playGame() method

Check out the comments in the code below for the playGame() method on the LogicGamePlayer class to get an idea of how the game player bot works.

public void playGame() {

    System.setProperty("webdriver.chrome.driver", "chromedriver");


    // Instantiate a WebDriver 
    WebDriver driver = new ChromeDriver();

    //Set up a waiter for the driver
    driver.manage().timeouts().implicitlyWait(10, TimeUnit.SECONDS);

    //The url that we want to navigate to
    String baseUrl = "http://htmlpreview.github.io/?https://github.com/jackpowers/The-Logic-Game/blob/master/gameboard/logicgame.html";
    driver.get(baseUrl);

    //Instantiate some DOM elements using xpath

    WebElement form1 = driver.findElement(By.xpath("//select[@id='1']"));
    WebElement form2 = driver.findElement(By.xpath("//select[@id='2']"));
    WebElement form3 = driver.findElement(By.xpath("//select[@id='3']"));

    WebElement form4 = driver.findElement(By.xpath("//select[@id='4']"));
    WebElement form5 = driver.findElement(By.xpath("//select[@id='5']"));
    WebElement form6 = driver.findElement(By.xpath("//select[@id='6']"));

    WebElement form7 = driver.findElement(By.xpath("//select[@id='7']"));
    WebElement form8 = driver.findElement(By.xpath("//select[@id='8']"));
    WebElement form9 = driver.findElement(By.xpath("//select[@id='9']"));

    WebElement tilde4 = driver.findElement(By.xpath("//button[@id='tilde4']"));
    WebElement tilde5 = driver.findElement(By.xpath("//button[@id='tilde5']"));

    WebElement winner = driver.findElement(By.xpath("//h2[@id='message']"));


    WebDriverWait wait = new WebDriverWait(driver, 20);

    //true player turn 1

    wait.until(ExpectedConditions.elementToBeClickable(form1));

    Select dropdown1 = new Select(form1);

    dropdown1.selectByIndex(1);

    //false player turn 1

    wait.until(ExpectedConditions.elementToBeClickable(form2));

    Select dropdown2 = new Select(form2);

    dropdown2.selectByIndex(2);

    //true player turn 2

    wait.until(ExpectedConditions.elementToBeClickable(form3));

    Select dropdown3 = new Select(form3);

    dropdown3.selectByIndex(3);

    //false player turn 2

    wait.until(ExpectedConditions.elementToBeClickable(form4));

    Select dropdown4 = new Select(form4);

    dropdown4.selectByIndex(4);

    //true player turn 3

    wait.until(ExpectedConditions.elementToBeClickable(form5));

    Select dropdown5 = new Select(form5);

    dropdown5.selectByIndex(1);

    // false player turn 3

    wait.until(ExpectedConditions.elementToBeClickable(tilde4));

    tilde4.click();

    // true player turn 4

    wait.until(ExpectedConditions.elementToBeClickable(tilde5));

    tilde5.click();

    // false player turn 4

    wait.until(ExpectedConditions.elementToBeClickable(form6));

    Select dropdown6 = new Select(form6);

    dropdown6.selectByIndex(2);

    // true player turn 5

     wait.until(ExpectedConditions.elementToBeClickable(form7));

    Select dropdown7 = new Select(form7);

    dropdown7.selectByIndex(3);

    // false player turn 5

    wait.until(ExpectedConditions.elementToBeClickable(form8));

    Select dropdown8 = new Select(form8);

    dropdown8.selectByIndex(4);

    //true player turn 6

    wait.until(ExpectedConditions.elementToBeClickable(form9));

    Select dropdown9 = new Select(form9);

    dropdown9.selectByIndex(1);

    System.out.println(winner.getText());


    driver.close();
            driver.quit();
}
