Getting Started with Protractor

This post is a quick introduction to Protractor, a Node.js QA automation tool for AngularJS.

First things first

For this demo, you should have Node.js and npm installed on your machine.

Protractor uses Jasmine. Read up on it.

You alsoneed a Selenium Server. You will need to have the Java Development Kit (JDK) installed to run the standalone Selenium Server. Verify this by running java -version from the command line.

You should also create a free account at Remember the Milk.

Install Protractor, Install Helper Files, and Fire Up a Server

Run the commands below in your terminal.

Global install of Protractor:
npm install -g protractor

Download some helpers:
webdriver-manager update

Start a Selenium Server:
webdriver-manager start

Let’s make a test!

Open another terminal window and create a new folder for the project.

We need two files in our folder: a config file and a spec file.

The config file points us to our Selenium Server, test framework, and spec file.

//conf.js
exports.config = {
framework: ‘jasmine’,
seleniumAddress: ‘http://localhost:4444/wd/hub',
specs: [‘spec.js’]
}

The spec file is where we set up the test we want to execute. Be sure to modify the code below to contain your Remember the Milk user name and password.


//spec.js
describe(‘Protractor Remember the Milk Demo’, function() {
it(‘should do a log-in’, function() {
//webpage that we navigate to
browser.get(‘https://www.rememberthemilk.com');
//Confirm the title
console.log(browser.getTitle());
expect(browser.getTitle()).toEqual(‘Remember The Milk: Online to-do list and task management’);
//Log-in
element(by.linkText(‘Log in’)).click();
element(by.id(‘username’)).sendKeys(“yourUserName”);
element(by.id(‘password’)).sendKeys(“yourPassword”);
element(by.id(‘login-button’)).click();
//Check for login to work
browser.refresh();
// https://github.com/angular/protractor/blob/master/docs/timeouts.md#waiting-for-angular-on-page-load
browser.waitForAngularEnabled(false);
});
});

Run it!

Use this command in your terminal window:

protractor conf.js

You should see a browser navigate to the Remember the Milk homepage and then log in.

The End

Stay tuned for more advanced Protractor demos in the future!
