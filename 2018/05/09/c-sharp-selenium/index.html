Selenium Testing with C#

A Selenium Demo in C

Thus far, I’ve only dabbled a bit in C#. I built this thing, but that’s about it. A new job that I’m about to start is heavy on C#, so I thought I’d put together a short demo app that tests some links on a website. For this demo, we’ll make sure that some of the ancient links on the infamous super-classic vintage climbing website, SuperTopo, are working as they should.

I built this app using Visual Studio Community Edition. Download it if you haven’t already.

Follow the steps below to build your own SuperTopo Selenium Demo App in C

1. Create a new MSTest Solution

Open Visual Studio and select File -> New Solution. In the pop-up window, select .Net Core -> Tests -> MSTest Project. Create a name for your project (I’m using SuperTopo1, so use this name if you want to copy and paste code.), and select a directory on your computer to store it.

2. Add Packages

Select Project -> Add NuGet Packages. Type in “Selenium” in the search field. Check the checkboxes next to Selenium.Webdriver and Selenium.Support. Click the Add Packages button and then accept the licenses in the pop-up window.
