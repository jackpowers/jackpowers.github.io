Getting Started with MQTT.js

An MQTT.js Demo App in Electron

MQTT.js is a client library for MQTT machine-to-machine protocol for Node.js.

This demo app uses Mosquitto MQTT Server/Broker to subscribe and publish to an MQTT topic linked to this temperature guage. Using the app, you can set a temperature for the guage via publishing to the MQTT topic. When you publish, you can see the guage move to the temperature you selected. The temperature that you selected will also appear in the topic’s messages displayed below the guage in a text box. Pressing the “Test” button will verify that the temperature you selected appears in the MQTT topic’s messages.
