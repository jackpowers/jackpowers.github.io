Getting Started with NightWatch.js!

Mountain-Project-NightWatch.js

Nightwatch.js is a Node.js based testing solution for browser based apps and websites. It uses the W3C WebDriver API to perform commands and assertions on DOM elements.

This NightWatch.js app tests the functionality of mountainproject.com. With an Amazon S3 account, you can store test data including screenshots on S3. The app can also be configured to use SauceLabs for testing on multiple browsers and simulated devices. I created this app with the help of this tutorial:

https://github.com/dwyl/learn-nightwatch

Welcome to AutomataHackers

Welcome to AutomataHackers!

My name is Jack. I am an automation engineer working in Minneapolis. You can check out some of my QA automation projects here on my GitHub.

AutomataHackers is a blog dedicated to QA automation. Modern software QA uses a suite of facinating tools to automatically control browsers and apps and compare their expected behavior with their actual behavior. These tools allow for rapid and resource-efficient validation of software functionality. In this blog, I’ll be exploring QA concepts, demo-ing projects, and writing about working in the QA automation space.
